# ACCA PathFinder Demo

A Streamlit prototype for an ACCA Campus Ambassador idea:
- BMBU exemption estimator
- Personalized ACCA exam roadmap
- General ACCA exam information
- ACCA webinars/events information page with mock fallback data

## How to run

```bash
pip install -r requirements.txt
streamlit run app.py
```

Then open the local Streamlit URL shown in the terminal.

## Notes

This is a demo for campus promotion and planning only.
- Exemption results are estimates, not official decisions.
- Official exemptions must be checked through ACCA Exemptions Calculator and ACCA review.
- Exam dates/events should be verified through ACCA official website and myACCA.
